# ses
基于 RBAC 简单实现动态菜单及权限管理系统

项目介绍：

* 基于 SpringBoot 

* 基于 JWT 实现前后分离,Spring Security 安全控制

* 前端采用 topjui （基于 easyui 的美化框架 ）

* 实现了三级动态菜单、页面按钮动态授权功能

* 实现了用户管理、菜单管理、角色管理

* 集成 Swagger 接口管理
